# Dorock's Mod
Dorock's Mod is a mod used for enhancing Minecraft. Not just the visual but also behind the scene.

## Here everything that the mod has or does:
- DTab (A tab in creative only used to group all blocks or items from this mod)
- DStick (Item used for testing)
- DBlock (Block used for testing)
- DebugGui (Gui used for debug)
- DorockDebug (Command used for showing DebugGui)
